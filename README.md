# Advanced-Excel-Project
This project analyzes supermarket sales data using Excel Dashboard.

## Supermarket Sales Analysis (Advance Excel Project)

## Overview  
This project explores supermarket sales data to understand customer behavior, product performance, and revenue trends.The final dashboard presents key insights using interactive charts and filters in Microsoft Excel.

## Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Charts (Bar, Pie, Line, Donut)  
- Slicers for Branch and Year  

## Project Workflow  
1. **Data Collection:** Imported raw supermarket sales data into Excel.  
2. **Data Cleaning:** Removed duplicates, fixed data types, and standardized product names.  
3. **Pivot Analysis:** Created summaries for sales, quantity, rating, and customer types.  
4. **Dashboard Design:** Built a visual dashboard with filters and dynamic charts.  

## Key Insights  
- **Top Product Line:** Electronic Accessories had the highest quantity sold.  
- **Best Performing Branch:** Branch A led in overall sales.  
- **Peak Month:** June recorded the highest revenue.  
- **Customer Type:** Balanced distribution between Member and Normal customers.  
- **Gender Split:** Male (499) and Female (501) customers.  
- **Popular Payment Method:** E-wallet was used most frequently.  
- **City Performance:** Yangon had the highest sales.
